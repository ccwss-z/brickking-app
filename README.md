# 砖王 PWA

这是砖王的 Web/PWA 版本，目标是绕开 iOS 安装签名限制，让手机直接通过 Safari 访问使用。

## 功能

- 上传完整游戏截图
- 浏览器本地定位 14 x 10 棋盘并画切割线
- 图鉴默认为空，不内置初始砖块
- 支持图鉴分类、添加分类、删除分类、改名、转移分类、单个删除、多选删除
- 支持直接批量导入砖块图，也支持截图分割后二次确认导入
- 支持上传截图前选择识别图鉴分类
- 支持识别校正、开始分析、步骤展示、自动下一步
- 支持添加到 iPhone 主屏幕

## 部署

静态文件可直接部署到 GitHub Pages。

```bash
cd web-pwa
python3 -m http.server 5173
```

手机访问 GitHub Pages 地址后，Safari 分享按钮里选择“添加到主屏幕”即可。

## 限制

- PWA 的算力来自手机浏览器，不使用 GitHub 服务器。
- 浏览器没有 iOS Vision 框架，因此识别算法是 Canvas 轻量版，准确率依赖图鉴完整度。
- 深度搜索会受手机浏览器性能限制。
